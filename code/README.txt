Exemplos do minicurso "Introdução a microcontroladores"
Microcontrolador: MSP430G2231

* Aula 2 - Módulo de Clock
ex02_01:
Este exemplo realiza a configuração dos registros relacionados ao clock, usando VLO como oscilador.
ex02_02:
Este exemplo realiza a configuração para uso do DCO como oscilador.
ex02_03:
Este exemplo realiza a configuração dos registros relacionados ao clock.

* Aula 3 - Módulo de Entrada e Saída
ex03_01:
Este exemplo realiza a configuração dos PORT's para piscar LED.
ex03_02:
Este exemplo possui por objetivo fazer LED piscar utilizando delay timer acionado a partir de um botão.

* Aula 4 - Módulo de Timer
ex04_01:
Este exemplo possui por objetivo fazer LED piscar utilizando interrupção de timer.
ex04_02:
Este exemplo possui por objetivo fazer LED piscar utilizando interrupção de timer e capture/compare.
ex04_03:
Este exemplo possui por objetivo fazer LED piscar utilizando interrupção de timer e capture/compare para gerar 3 sinais com frequências diferentes.
